# web3-provider-patchers
This repository demonstrates how to create a custom Web3 provider using the web3-provider-engine library and apply patchers from web3-provider-patchers for enhanced Ethereum interaction.

## Prerequisites

- Node.js (>=12.0.0)
- npm (Node Package Manager)

## Getting Started

1. Clone the repository:
```git clone https://github.com/yourusername/web3-provider-patcher-demo.git```

2. Navigate to the project directory:
```cd web3-provider-patcher-demo```

4. Install dependencies:
```npm install```

5. Configure Infura Credentials:
Replace `YOUR_INFURA_PROJECT_ID` and `YOUR_INFURA_PROJECT_SECRET` in the main.js file with your actual Infura project credentials.

5. Run the demo script:
```node main.js```

## Features
Custom Web3 provider setup using web3-provider-engine.
Integration of web3-provider-patchers for filtering and wallet interaction.
Basic Ethereum interactions and subscription to new block headers.
